# プロジェクト概要

## 目的
- 中古PC・パーツ・周辺機器の在庫/売買をブラウザで管理する
- 仕入価格・販売価格から利益と在庫金額を可視化する

## 仕様（現状）
- 技術: Next.js App Router、React、TypeScript、Tailwind CSS v4
- データモデル: `Item` / `Category` / `ItemStatus` / `Summary`
- 画面構成:
  - サマリーカード（在庫数/販売数/利益合計/在庫金額）
  - ステータス絞り込み（すべて/在庫/販売済み）
  - 在庫一覧テーブル
  - 追加/編集モーダル
- 入力項目: 商品名、カテゴリ、状態、仕入価格、仕入日、販売価格、販売日、メモ
- データ保存: `localStorage` に保存（`inventory-items-v1`）
- 起動: `npm run dev` は `next dev --webpack` 固定

## 進捗
- UIと基本ロジック（一覧表示・集計・追加/編集/削除）実装済み
- 文字化け/構文破損を修復済み
- `localStorage` でデータ永続化を追加済み

## 残タスク（候補）
- 検索・並び替え・フィルタ追加
- CSV入出力
- データのクラウド保存（API/DB）
- 認証/ユーザー管理
- テスト追加
